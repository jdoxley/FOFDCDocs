Start the promotion process with a skeleton promotion object.

The promotion parameter requires a skeleton Promotion object that has at least the following properties with values:

Property Description
Promotion.PromotionFor A Trooper represeting who the promotion is for.
Promotion.CurrentBoard A PromotionBoardLevel that indicates the starting board of this promotion.
Promotion.PromoteFrom A int that represents the starting rank of this promotion.
Promotion.PromoteTo A int of a rank that indicates the goal of this promotion.
Promotion.Reason A string that details the reason behind this promotion.
Promotion.Forced A bool that indicates if this promotion was forced or not.
Parameters
invokerThe ClaimsPrincipal of the Trooper who started the promotion.
promotionA skeleton Promotion object.
Returns
